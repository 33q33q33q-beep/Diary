# 日記アプリのベース場所

このフォルダが **本体の実体** です（Git リポジトリ含む）。

## ディスク上のパス

```
~/Library/Mobile Documents/com~apple~CloudDocs/ClaudeCode/Diary
```

Finder では **iCloud Drive → ClaudeCode → Diary** に相当します。

## Cursor ワークスペース側

`~/ClaudeCode/Diary` は上記フォルダへの **シンボリックリンク** になっています。  
いつもどおり `ClaudeCode` を開けば同じ内容を編集できます。

※ macOS に標準の `/iCloud/...` というパスはないため、iCloud 上の実体は上記「Mobile Documents」配下になります。
