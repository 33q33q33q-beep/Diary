☁️ クラウド同期設定

同期の方式はこの下のプルダウンです（サイドバーには出ません）。
Gist／Googleドライブ／ローカルのみから選べます。


GitHub Gist

手順：
1. GitHub → Settings → Personal access tokens
2. Note に「diary」と入力、Expiration: No expiration
3. Scopes で gist にチェック → Generate token
4. 表示されたトークンをコピーして下に貼り付け

※ 初回はGist IDを空欄にすると自動作成されます

📱 iPhoneなど他の端末でも使う場合：「設定リンクをコピー」→ Safariで開くと自動設定されます

Googleドライブ

Google Cloud Console で「OAuth クライアント ID（ウェブアプリ）」を作成し、承認済みの JavaScript 生成元にこのサイトのURL（例: https://あなた.github.io）を追加してください。スコープはアプリが作成したファイルのみ（drive.file）です。

📄 マニュアル：同期のたびに、使い方 USER-MANUAL.pdf をフォルダ LifeTimeDiary の直下へ置きます。Googleドライブのアプリからそのまま開けます（公開サイトに PDF がある場合のみ）。

重要（iPhone）
クライアント ID を Safari のアドレス欄にだけ貼らないでください。ブラウザが勝手に http://数字-… と解釈して Google の 404 になります。
手順: ①下の欄に貼る または ②「Drive設定リンク」をコピーした全体の https://… URLをアドレス欄で開く。

📱 Macで「Drive設定リンクをコピー」→ iPhoneのSafariでリンク全体を開くと ID が入ります。

クラウド自動同期は使いません。サイドバーの 「ZIP書出し」 で LifeTimeDiary/ 形式のZIPを作り、iPhoneのファイルアプリ経由でiCloudに保存してください。「ZIP読込」で戻せます。
